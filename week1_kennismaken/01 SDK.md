## SDK installatie

> `sdk` staat voor `software developers kit`

Ga naar:
- https://dotnet.microsoft.com/en-us/download/dotnet/8.0

- installeer de SDK 8.0.x (hoogste) voor jouw systeem
    - voor windows zal dat de x64 versie zijn
- open je terminal/cmd
    - type dotnet
